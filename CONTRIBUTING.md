# Contributing

  ValoonSync is **closed-source software**. This repository only hosts
  official releases, so we do not accept pull requests here.

  However, you can still contribute:

  - 🐛 **Report bugs** — open an issue with reproduction steps.
  - 💡 **Suggest features** — open an issue describing your use case.
  - 🌍 **Translations** — found a typo or awkward phrasing in English,
    Español, Deutsch, or Français? Let us know in an issue.

  Thank you for helping make ValoonSync better!
