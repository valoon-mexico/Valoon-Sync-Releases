# Security Policy

  ## Supported Versions

  Only the **latest release** of ValoonSync receives security updates.
  Please make sure you are running the most recent version — the app updates
  automatically, but you can always download the latest installer from the
  [Releases page](https://github.com/valoon-mexico/Valoon-Sync-Releases/releases/latest).

  | Version | Supported |
  | ------- | --------- |
  | Latest  | ✅         |
  | Older   | ❌         |

  ## Reporting a Vulnerability

  We take security seriously. If you discover a vulnerability in ValoonSync
  or any Valoon service, please report it **privately** — do not open a
  public issue.

  - 📧 Email: **support@valoon.chat**
  - Include: a description of the issue, steps to reproduce, affected version,
    and any relevant logs or screenshots.

  ### What to expect

  - We will acknowledge your report within **3 business days**.
  - We will keep you informed of the progress toward a fix.
  - Once resolved, we will credit you in the release notes (unless you prefer
    to remain anonymous).

  Please give us a reasonable time to address the issue before any public
  disclosure. Thank you for helping keep Valoon users safe. 🙏
