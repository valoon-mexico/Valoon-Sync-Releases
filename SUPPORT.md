# Support

  ## 🐛 Found a bug?

  The best way to report a problem is to attach a **diagnostic report** —
  ValoonSync can generate one for you with everything we need to investigate.

  ### How to generate a diagnostic report

  1. Open the report dialog:
     - If a sync just failed, click the **"Report a problem"** button on the
       main screen, **or**
     - Go to **Settings → Support → "Export diagnostics"** (available anytime).
  2. Review the list of recent failures and, optionally, describe what
     happened (e.g. *"two photos from yesterday are missing"*).
  3. Click **"Generate report"**.

  ValoonSync will create a **ZIP file** containing logs and details of the
  failed files, open the folder with the file selected, and copy a short
  summary to your clipboard.

  > 💡 You can find previously generated reports anytime via
  > **Settings → Support → "Open reports folder"**.

  ### Where to send it

  Email the ZIP to **support@valoon.chat**, including:

  - The generated report ZIP (attached)
  - Your ValoonSync version (visible in Settings)
  - Your Windows version
  - What you were doing when the problem occurred

  You can also open an issue on this repository with the same information —
  but please **do not attach the diagnostic ZIP to public issues**, since it
  may contain file names from your projects. Send it by email instead.
